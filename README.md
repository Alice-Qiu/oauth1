# OAuth1

Simple OAuth 1.0 helper. Take a look at the specs on how to use it.

## Installation

Add this line to your application's Gemfile:

    gem 'oauth1'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install oauth1
